# Image Classification Using CNN (Cats vs Dogs)

This project classifies images of cats and dogs using a CNN model and deploys it as a web app.

## How to Run Locally
1. Install dependencies  
   pip install -r requirements.txt

2. Train model  
   Open `notebook/Cats_vs_Dogs_CNN.ipynb` and run all cells

3. Run web app  
   cd app  
   streamlit run app.py

## Hosting
This project can be hosted on:
- Streamlit Cloud
- HuggingFace Spaces
- Render
